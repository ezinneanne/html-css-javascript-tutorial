CSS Typography And The Box model

Now, you will cover the core concepts of typography in css and the concept of the box model.

Typography​

Typography simply has to do with how to organise your fonts and text to give a great aesthetic appeal. Typography is applied in css and you will cover the properties of typography in this session.

Properties of typography are font-size, font-family, font-weight, font-style, line-spacing, word-spacing, text-align, and text-decoration.

Font-size​

Has to do with the size of the text. You covered font-size in the previous session.

Font-weight​

Font weight is used to determine the thickness or lightness of the text.

For example, 

font-weight: bold;

Makes a font bolder. You could give font-weight using numbers too.

Example,

font-weight: 600;

Will also make the font bolder.

Font-style​

Font-style is used to determine the styling of the text. font-style:normal; makes the text normal while font-style:italic; makes the text slanted.

Font-family​

Font-family is used to select a desired font for the text. When using font family always add a default or fallback values or the browser would choose a fallback value for you. Unless you are using a fallback value already.

Example:

body {
  font-family: 'Roboto', sans-serif;

sans-serif is a fallback value. To have access to fonts like Roboto, use google fonts.

Letter-spacing and word-spacing​

Letter spacing is used to give equal spacing between lines while word spacing is used to give equal spacing between words.

Example:

body {
  letter-spacing: 2px;
  word-spacing: 4px;
}

Text-align​

Text-align is used to align the position of the text. Whether to position the text right, left or centre. It is written like this:

body {
  text-align: centre;
}

This would align the text to the centre.

Text-decoration​

Text-decoration is used to underline or remove underline from a text.

For example:

body {
  text-decoration: underline;
}

h1 {
  text-decoration: none;
}

Box model​

The box model is a concept of how css styles html elements. Each element is like a box and this box is a rectangular-shaped item. Each part of the box comprises these properties: margin, border, padding, width and height.

The box model is an important part of css because understanding the box model would make it easier for you to style web pages.

The image above, is an illustration of the box model. It has a top, bottom, left and right surrounding the box.

Margin is the outer body surrounding an element.

The border is in between the margin and padding while the padding is the inner body.

Example:

index.html

  <!DOCTYPE html>
  <html>
    <head>
      <meta charset=utf-8"/>
      <title>A simple html page</title>
      <link rel="stylesheet" href="styles.css"/>
    </head>
    <body>
      <h1>Building A Web Page</h1>
      <p class="page">How To Design A Webpage</p>
      <ul>
         <li>Colors</li>
         <li>Typography</li>
         <li>Selectors</li>
         <li>Box model</li>
       </ul>
    </body>
  </html>

styles.css

  h1 {
    margin: 20px;
    padding: 20px;
    border: 3px solid blue;
  }

  p {
    margin: 35px;
    border: 2px solid green;

The measurements are applied all around. If you wanted to style only the left edges, you would use margin-left or border-left`, the same thing happens on all the edges.

If you have a Firefox browser, click on three button icon and select Developer Tools and when it opens you can access the elements on your webpage and see how you can work with them.
