Media queries, flexbox and css grid

This is the last part of the css session. Congrats on making it this far! In this session, you will cover media queries, flexbox and css grid.

Media queries​

Media queries are used to make web pages responsive. When a webpage is responsive, it means you can render the web page on any device.

Media queries were introduced in css3.

When using media queries, you can use them as all, print, and screen.

  • all - used for all media types
  • print - only for printers
  • screen - for devices with screens like computers, tablets and phones.

Example

index.html

  <!DOCTYPE html>
  <html>
    <head>
      <meta charset=utf-8"/>
      <title>A simple html page</title>
      <link rel="stylesheet" href="styles.css"/>
    </head>

This will make the body element change when the viewport is 600px or more.

Flexbox​

In this session, you will cover flexbox which means Flexible Box Module and all its concepts.

Flexbox is a layout model on CSS that makes it easier to position items on a webpage depending on the screen size. Flexbox replaced float which was a previous technique to position items on a webpage.

To use flexbox, you would need to set the display property to flex. The parent the flex is applied to is called container and the children is called the items.

Example,

index.html

<!DOCTYPE html>
  <html>
    <head>
      <meta charset=utf-8"/>
      <title>A simple html page</title>
      <link rel="stylesheet" href="styles.css"/>
    </head>
    <body>
      <h1>Building A Web Page</h1>
      <div class="flex-container">
      <p>Lorem ipsum dolor sit amet, illum definitiones no quo, maluisset concludaturque et eum, altera fabulas ut quo. Atqui causae gloriatur ius te, id agam omnis evertitur eum. Affert laboramus repudiandae nec et. Inciderint efficiantur his ad. Eum no molestiae voluptatibus.

styles.css

.flex-container {
  display: flex;

When you check your browser, you would discover flex had lined up the child elements because they are inheriting that flex property from the div.

There are properties of flex which includes: flex-direction, flex-wrap,justify-content and so on which are not covered in this tutorial. Check css tricks for more details on flexbox.

CSS Grid​

Now, we have come to the last session of CSS.

Grid is an important concept in CSS. Just like Flexbox, grid is a layout model of rows and columns in a grid system. Which is used to design web pages. Grid also replaced floats as a positioning technique.

A grid layout consist of the parent element which is the container while the child elements are the items.

Example,

index.html

  <!DOCTYPE html>
  <html>
    <head>
      <meta charset=utf-8"/>
      <title>A simple html page</title>
      <link rel="stylesheet" href="styles.css"/>
    </head>
    <body>
      <h1>Building A Web Page</h1>
      <div class="grid-container">
      <p>Lorem ipsum dolor sit amet, illum definitiones no quo, maluisset concludaturque et eum, altera fabulas ut quo. Atqui causae gloriatur ius te, id agam omnis evertitur eum. Affert laboramus repudiandae nec et. Inciderint efficiantur his ad. Eum no molestiae voluptatibus.
</p>

    </div>

styles.css

.grid-container {
  display: grid;
}
