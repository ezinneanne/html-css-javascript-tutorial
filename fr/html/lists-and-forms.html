Character References, Lists and Forms

Character References​

These are entities used to represent certain symbols in html. They are: The white space character (&nbsp;), in html you could only give one white space even if you give more . It is ignored in the browser. So the white space character is what you use to denote that.

<p>Create   more spaces &nbsp; here only</p>

The greater than (&gt;) or less than (&lt;) characters These symbols are used in html as tags so if you used them like that, the browser would assume you are writing a html tag. So you use the characters when you want to write those symbols.

For example:

<p>3 &gt; 2 &lt; 4</p>

Other characters include; &amp; for ampersand(&), &copy; for copy(©) and so on. Check this list of all character references in HTML for more.

Lists​

Lists in Html are used to group items or organize contents. There are two popular types of lists which are: unordered and ordered lists.

Unordered Lists​

Unordered Lists are used to group items that you do not wish to number. They are usually represented in bullets, circles or square.

For example:

<p>Reasons to learn how to code</p>
<ul>
  <li>It is fun</li>
  <li>You will make money</p>
  <li>It would take you to great places</li>
<ul>

The li is used to denote the items while the ul indicates that it is an unordered list.

Ordered Lists​

Ordered Lists are used to group items in a numerical or alphabetical order. Ordered lists are used for hierarchy purpose. When writing a set of items that must follow each other then use ordered lists. Also use ordered lists for organizations of items.

For example:

<p>How to fry an egg</p>
<ol>
  <li>Pour oil to heat</li>
  <li>Pour egg</li>
  <li>Wait for it to fry</li>
</ol>

You could also use it for other instances, like calling a list of names.

Nested Lists​

Nested lists are lists stacked in each other. You could be writing a program and you would to use lists both ordered and unordered list. So that is how nested lists came to be. For example,

<ul>
  <li>Things I do everyday:</li>
    <ol>
      <li>Pray</li>
      <li>Read</li>
      <li>Write Code</li>
    </ol>
  <li>Things I wish I do every day:</li>
    <ol>
      <li>Watch movies</li>
      <li>Sleep</li>
    </ol>
</ul>

When using ordered lists, numbers are used as the default but if you want alphabets then you can specify with the list-style-type command. Example:

<ol list-style-type=alphabet>
  <li>Watch movies</li>
  <li></li>
</ol>

Html Forms​

Html forms are used to fill in user data and process the information.

<form>

</form>

Input buttons have several types: text for simple text. numbers for digits. email for email. You would need to specify the type.

These buttons also require a label element for accessibility. Screen readers usually read label elements so it's best practice to add them. *Note: The for attribute should contain the same info as your id attribute*

<input type="text" id="your_name" name="user name"  value="your name"/>
<label for="your_name">Your Name</label

<input type="numbers" id="phone_number" name="phone number" value="your phone number"/>
<label for="phone_number">Your Phone Number</label>

<input type="email" id="email" name="email" value="your email"/>
<label for="email">Your Email</label>

There are also input buttons for checkboxes and radio buttons. 

Gender:
<input type="radio" id="male" name="gender_type" value="Male"/>
<label for="male">Male</label>
