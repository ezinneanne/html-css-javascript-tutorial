Html, CSS And JavaScript Tutorial

Brief Introduction​

Web development is the process of building websites. Building a website requires technical knowledge. Markup languages like html and css are paramount in building web pages.

Web design explained​

Html lays the structure of the web page while CSS styles the page and makes it presentable. Then programming languages are used to make websites dynamic. When you register your details on a site and the site saves it and sends a welcome message with your name. That is the work of programming languages. These languages are used to interact with the web without the programming languages, websites would be static.

JavaScript is a programming language used in building websites. JavaScript was built primarily for building websites and it has advanced with great features, frameworks and libraries.

In this tutorial, you will learn important topics on Html, CSS And JavaScript and you will build an application at the end of this tutorial.

Prerequisites​

To follow properly through the sections of this tutorial, you will need:

  • A web browser like Chrome or Firefox.
  • A text editor like vscode.
