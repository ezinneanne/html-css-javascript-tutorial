Headings, Paragraphs and Images

Headings​

So, let's talk about the headings. The headings have a hierarchical structure with h1 being the highest and h6 being the smallest.

For example:

<h1>This is the first</h1>
<h2>This is the second</h2>
<h3>This is the third</h3>
<h4>This is the fourth</h4>
<h5>This is the fifth</h5>
<h6>This is the sixth</h6>

When you enter this in your browser, it should show the hierarchical structure of the headings. You use the headings when you want to create a title or subtitles.

Paragraphs​

Paragraphs are used to create paragraphs or begin a sentence on a new line. So let's create a paragraph:

<p>This is a paragraph</p>

Now let's move to other html tags

Image basics​

Now, let us create images. You could see images on every webpage and they bring out the asthetics in every page. Images require a source attribute in it, which is denoted by the src. Images also require an alt attribute which means alternative to describe the image content.

To create images, enter:

<img src='gif.png' alt='a young girl looking at the sun'/>

Images in a different directory and from the web​

The previous image I used, I assumed it to be in the same directory as the index.html file. But what if it was in a different directory? If the gif.png file was in a different directory say images. Then to display the image, you would do:

<img src='./images/gif.png' alt='a young girl looking at the sun'/>

To display an image from the web directly on the webpage, you would do: Copy the image address and paste it in to the src attribute.

<img src='www.google.com/gif.png' alt='a young girl looking at the sun'/>

Width and Height​

Images have width and height attributes which you can add to them while writing your code. You could work on the previous image and give it specific height and weight values.

<img src='./images/gif.png' alt='a young girl looking at the sun' width='16' height='42'/>
